## 📌 Project Overview
This is an AI-powered ATS Resume Analyzer and Recruitment Intelligence platform built using **Python** and **Streamlit**. The application automates the candidate screening process for hiring teams. It reads uploaded PDF resumes, extracts skills, flags upskilling gaps, and displays visual intake metrics on an interactive Admin Dashboard.

---

## ✨ Main Features
- **Dynamic Text Parsing:** Automatically extracts raw text lines and total page count directly from uploaded PDF documents.
- **Contact Information Mining:** Uses Regular Expressions (RegEx) to instantly discover and display emails and contact numbers.
- **Case-Insensitive Skill Matching:** Cross-references resume text to identify technical keywords (e.g., Python, SQL, DSA, Java).
- **AI Skill Gap Evaluation:** Auto-detects missing core tools (like Docker, Kubernetes, AWS Cloud) and suggests learning paths.
- **Enterprise Analytics Dashboard:** Transforms application counts into live visual distributions using dynamic Bar and Line charts.
